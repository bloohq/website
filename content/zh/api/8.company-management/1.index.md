---
title: 公司管理
description: Blue API 中公司管理操作的概述。
---

Blue API 提供全面的公司管理功能，允许您在 Blue 工作区内创建、查询和管理公司。

## 可用操作

### 创建公司
- [创建公司](/api/company-management/create-company) - 使用自定义名称和别名创建新公司

### 查询公司
- [查询公司](/api/company-management/query-company) - 使用 ID 或别名检索公司信息

## 关键概念

### 公司识别
在 Blue 中，可以使用两种方法识别公司：
1. **公司 ID** - 创建公司时自动生成的唯一标识符 (CUID)
2. **公司别名** - 创建公司时您指定的 URL 友好标识符

### 访问控制
- 用户只能访问他们是成员的公司
- 公司操作遵循用户权限和访问级别
- 被禁止的公司无法通过 API 访问

### 计费考虑
Blue 中的每家公司都是单独计费的，因此创建新公司可能会影响您的账户计费。
