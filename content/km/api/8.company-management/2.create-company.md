description: របៀបបង្កើតក្រុមហ៊ុនថ្មីមួយដោយប្រើ Blue API។
---

ដើម្បីបង្កើតក្រុមហ៊ុនថ្មីមួយ អ្នកអាចប្រើ mutation ខាងក្រោម៖

```graphql
mutation CreateCompany {
  createCompany(input: { name: "new", slug: "new-slug" }) {
    id
  }
}
```

សូមចំណាំថាការបង្កើតក្រុមហ៊ុនថ្មីអាចមានឥទ្ធិពលលើការបង់ប្រាក់នៃគណនី Blue របស់អ្នក ពីព្រោះក្រុមហ៊ុននីមួយៗត្រូវបានគិតថ្លៃដោយឡែក។

នេះគឺជាជម្រើសសម្រាប់ mutation `createCompany` និងការពិពណ៌នារបស់ពួកវា៖

| ជម្រើស | ប្រភេទ | ការពិពណ៌នា |
|--------|------|-------------|
| name | String | ឈ្មោះនៃក្រុមហ៊ុនថ្មី។ នេះគឺជាផ្នែកដែលត្រូវការពិតប្រាកដ។ |  |
| slug | String | កំណែដែលឆបគ្នាជាមួយ URL នៃឈ្មោះក្រុមហ៊ុន។ នេះត្រូវបានប្រើនៅក្នុង URL របស់ក្រុមហ៊ុន ហើយគួរតែមានលក្ខណៈឯកត្តសាស្ត្រ។ នេះគឺជាផ្នែកដែលត្រូវការពិតប្រាកដ ហើយនឹងធ្វើជាអត្តសញ្ញាណឯកត្តសាស្ត្រសម្រាប់ក្រុមហ៊ុន។ |
