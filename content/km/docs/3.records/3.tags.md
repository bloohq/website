Tags គឺជាវិធីមួយដ៏មានប្រយោជន៍ក្នុងការរៀបចំការងាររបស់អ្នក។ អ្នកអាចប្រើពួកវាដើម្បីបង្ហាញប្រភេទកិច្ចការផ្សេងៗ កម្រិតបន្ទាន់ ឬប្រភេទណាមួយដែលសមស្របសម្រាប់គម្រោងរបស់អ្នក។ tag នីមួយៗអាចកំណត់ពណ៌សម្រាប់ការសម្គាល់ងាយស្រួល និងដើម្បីបន្ថែមពណ៌មួយចំនួនទៅកាន់កន្លែងការងាររបស់អ្នក។

## ការប្រើប្រាស់ Tags

- **ការបន្ថែម Tags**: ដើម្បីបង្កើត tag មួយ សូមពិចារណាអំពីទិដ្ឋភាពនៃកិច្ចការរបស់អ្នកដែលត្រូវការការចាត់ថ្នាក់—មិនថាវាជាលក្ខណៈនៃកិច្ចការ ដូចជា 'bug', 'back-end' ឬ 'UI' ឬអាទិភាពរបស់វា។ នៅពេលដែលអ្នកបានកំណត់ប្រភេទរបស់អ្នករួចហើយ អ្នកអាចបង្កើត tags បានយ៉ាងងាយស្រួល និងកំណត់ពណ៌តែមួយគត់សម្រាប់ការទទួលស្គាល់ភ្លាមៗ។
- **ការត្រៀបតាម Tags**: សមត្ថភាពត្រៀបរបស់ Blue ធ្វើឱ្យការគ្រប់គ្រងបន្ទុកការងាររបស់អ្នកកាន់តែមានជីវជាតិ។ អ្នកអាចត្រៀប tags ដោយប្រើ drop-down Filter icon ឬដោយផ្ទាល់នៅក្នុង board view ដោយចុចលើ tag ដែលចង់បាន។
