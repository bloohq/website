វាលខូសថមប្រកបដោយឯកសារអនុញ្ញាតឱ្យអាប់ឡូតនិងភ្ជាប់ឯកសារដោយផ្ទាល់ទៅកាន់កំណត់ត្រានៅក្នុង Blue។ ប្រភេទឯកសារស្ទើរតែគ្រប់មុខត្រូវបានគាំទ្រ រួមទាំងឯកសារ តារាងគណនា រូបភាព PDF និងផ្សេងៗទៀត។

![File 1](/docs/File_1.png "File 1")

ការអាប់ឡូតឯកសារដែលមានរចនាសម្ព័ន្ធធានាថាគ្រប់គ្នាទាំងអស់មានភាពមើលឃើញលើឯកសារដែលពាក់ព័ន្ធ

វាលឯកសារនីមួយៗអនុញ្ញាតឱ្យអាប់ឡូតឯកសារច្រើន ដោយឯកសារនីមួយៗមានទំហំទៅដល់ 5GB។ ឯកសារដែលបានអាប់ឡូត ដូចជារូបភាព PDF និងឯកសារ Word អាចត្រូវបានមើលជាមុនប្រសិនបើប្រភេទឯកសារត្រូវបានគាំទ្រ។ អ្នកប្រើប្រាស់គម្រោងទាំងអស់អាចទាញយកឯកសារដែលបានភ្ជាប់ដើម្បីចូលប្រើព័ត៌មានកំណត់ត្រានិងមាតិកាដែលពាក់ព័ន្ធ។

![File 2](/docs/File_2.png "File 2")

អ្នកប្រើប្រាស់អាចមើលជាមុននិងទាញយកឯកសារនៅក្នុងវាលខូសថមប្រកបដោយឯកសារ

ឯកសារដែលបានអាប់ឡូតត្រូវបានចែករំលែកទូទាំងគម្រោងនៅក្រោមផ្ទាំង ['ឯកសារ'](/km/docs/projects/file-management)។ ទោះជាយ៉ាងណាក៏ដោយ តួនាទីអ្នកប្រើប្រាស់ខូសថមប្រកបដោយការចូលប្រើកំណត់ត្រាដែលមានកំណត់នឹងមិនឃើញឯកសារទាំងនោះទេ។ ផ្ទាំងឯកសារគម្រោងគាំទ្រការបង្កើតថតដើម្បីរៀបចំឯកសារ ប៉ុន្តែឯកសារមិនអាចត្រូវបានតម្រៀបទៅក្នុងថតទាំងនេះពីក្នុងវាលខូសថមប្រកបនោះផ្ទាល់ទេ។

## ការរួមបញ្ចូលទម្រង់

វាលខូសថមប្រកបដោយឯកសារ [រួមបញ្ចូលយ៉ាងរលូនជាមួយទម្រង់](/km/docs/projects/forms) ដោយអនុញ្ញាតឱ្យអ្នកប្រើប្រាស់ខាងក្រៅអាប់ឡូតឯកសារនៅពេលដាក់ស្នើព័ត៌មាន។ ឧទាហរណ៍ បេក្ខជនអាចភ្ជាប់ CV និងសំបុត្រគម្របទៅទម្រង់ដាក់ពាក្យស្នើការងារ អតិថិជនអាចផ្តល់រូបភាពនិងឧទាហរណ៍សម្រាប់សំណើគម្រោងច្នៃប្រឌិត ឬកូនក្តីអាចដាក់ស្នើឯកសារនិងឯកសារចាំបាច់តាមរយៈទម្រង់ទទួលកូនក្តី។ តាមរយៈការរួមបញ្ចូលវាលខូសថមប្រកបដោយឯកសារលើទម្រង់ អ្នកពាក់ព័ន្ធខាងក្រៅអាចផ្តល់ឯកសារនិងដើមទុនសំខាន់ៗដើម្បីបំពេញកំណត់ត្រាដែលបង្កើតពីការដាក់ស្នើទម្រង់។ នេះផ្តល់អត្ថប្រយោជន៍នៃមាតិកាដែលប្រមូលបានដោយមិនតម្រូវឱ្យអ្នកប្រើប្រាស់ខាងក្រៅមានគណនីនៅក្នុង Blue។

## ការចូលប្រើឯកសារ API

ដូចឯកសារប្រព័ន្ធផ្សេងទៀត ឯកសារដែលបានអាប់ឡូតតាមរយៈវាលខូសថមប្រកបដោយឯកសារអាចត្រូវបានចូលប្រើនិងគ្រប់គ្រងតាមរយៈ [Blue API](/km/docs/integrations/api)។ នៅពេលឯកសារត្រូវបានភ្ជាប់ទៅកាន់កំណត់ត្រាដោយប្រើវាលឯកសារ ឯកសារទាំងនោះក្លាយជាអាចប្រើបានតាមរយៈការហៅ API ស្តង់ដារដែលពាក់ព័ន្ធនឹងឯកសារ។ អ្នកអភិវឌ្ឍន៍អាចប្រើប្រាស់សមត្ថភាព API ពេញលេញដើម្បីទាញយក អាប់ឡូត រៀបចំ និងរួមបញ្ចូលឯកសារពីវាលខូសថមប្រកបដោយឯកសារតាមកម្មវិធី។ មិនថាការដំណើរការដើមទុនដែលបានអាប់ឡូតដោយស្វ័យប្រវត្តិ ឬការបន្ថែម metadata នោះទេ API អនុញ្ញាតឱ្យមានការរួមបញ្ចូលយ៉ាងរលូនដោយផ្តល់ការចូលប្រើរួមបញ្ចូលទៅកាន់ឯកសារទាំងអស់ដែលបានបន្ថែមដោយអ្នកប្រើប្រាស់នៅក្នុង Blue។

តាមរយៈការភ្ជាប់ឯកសារនិងដើមទុនដែលត្រូវបានប្រើញឹកញាប់ វាលឯកសារលុបបំបាត់ការស្វែងរកតាមទីតាំងច្រើនដើម្បីរកមាតិកាដែលភ្ជាប់ទៅកាន់កំណត់ត្រា។ នេះផ្តល់ការចូលប្រើរហ័សនិងកណ្តាលគល់ទៅកាន់ឯកសារដែលអ្នកត្រូវការ។
