---

Om bedrijfsinformatie op te halen, kun je de `company` query gebruiken met ofwel de bedrijfs-ID of slug:

## Opvragen op Bedrijfs-ID

```graphql
query GetCompanyById {
  company(id: "company-id-here") {
    id
    name
    slug
    createdAt
    updatedAt
  }
}
```

## Opvragen op Bedrijfs-Slug

Dezelfde query accepteert ook een bedrijfs-slug, waardoor het eenvoudig is om bedrijfsinformatie op te halen met de URL-vriendelijke identifier:

```graphql
query GetCompanyBySlug {
  company(id: "company-slug-here") {
    id
    name
    slug
    createdAt
    updatedAt
  }
}
```

## Voorbeeld van een Respons

Beide queries zullen hetzelfde bedrijfsobject retourneren:

```json
{
  "data": {
    "company": {
      "id": "cuid123456789",
      "name": "Acme Corporation",
      "slug": "acme-corp",
      "createdAt": "2024-01-15T10:30:00.000Z",
      "updatedAt": "2024-03-20T14:45:00.000Z"
    }
  }
}
```

## Belangrijke Opmerkingen

- De `id` parameter accepteert zowel bedrijfs-ID's als slugs
- Alleen bedrijven waarvan de huidige gebruiker lid is, zullen worden geretourneerd
- Als het bedrijf niet wordt gevonden of de gebruiker geen toegang heeft, zal er een `CompanyNotFoundError` worden gegooid
- Verbannen bedrijven zullen een `CompanyBannedError` gooien
