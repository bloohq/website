De Blue API biedt uitgebreide mogelijkheden voor bedrijfsbeheer, waarmee je bedrijven kunt creëren, opvragen en beheren binnen je Blue-werkruimte.

## Beschikbare Operaties

### Bedrijven Creëren
- [Creëer een Bedrijf](/api/company-management/create-company) - Maak nieuwe bedrijven met aangepaste namen en slugs

### Bedrijven Opvragen
- [Vraag Bedrijf op](/api/company-management/query-company) - Haal bedrijfsinformatie op met behulp van ID of slug

## Belangrijke Concepten

### Bedrijfsidentificatie
Bedrijven in Blue kunnen op twee manieren worden geïdentificeerd:
1. **Bedrijfs-ID** - Een unieke identificator (CUID) die automatisch wordt gegenereerd bij het creëren van een bedrijf
2. **Bedrijfs-Slug** - Een URL-vriendelijke identificator die je opgeeft bij het creëren van een bedrijf

### Toegangscontrole
- Gebruikers kunnen alleen toegang krijgen tot bedrijven waar ze lid van zijn
- Bedrijfsoperaties respecteren gebruikersrechten en toegangslevels
- Verboden bedrijven kunnen niet via de API worden benaderd

### Factureringsoverwegingen
Elk bedrijf in Blue wordt afzonderlijk gefactureerd, dus het creëren van nieuwe bedrijven kan invloed hebben op de facturering van je account.
