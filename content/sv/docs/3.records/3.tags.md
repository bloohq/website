Taggar är ett användbart sätt att organisera ditt arbete. Du kan använda dem för att visa olika uppgiftstyper, brådska eller vilken kategori som helst som passar ditt projekt. Varje tagg kan färgkodas för enkel identifiering och för att lägga till lite färg till din arbetsyta.

<youtube url="https://www.youtube.com/watch?v=nlyY3cUUpkQ" />


## Använda Taggar

- **Lägga till Taggar**: För att skapa en tagg, överväg aspekterna av dina uppgifter som behöver kategorisering—vare sig det är uppgiftens natur, såsom 'bugg', 'back-end' eller 'UI', eller dess prioritet. När du har identifierat dina kategorier kan du enkelt skapa taggar och tilldela dem unika färger för omedelbar igenkänning.
- **Filtrera med Taggar**: Blues filtreringsfunktioner gör det mer intuitivt att hantera din arbetsbelastning. Du kan filtrera taggar med hjälp av rullgardinsmenyn Filter-ikonen eller direkt i tavelvyn genom att klicka på önskad tagg.

Taggning över Projekt <br/>
För organisationsövergripande vyer såsom Dashboards och kartor blir taggning särskilt viktigt eftersom Blue automatiskt slår samman taggar med samma namn. Så du behöver bara filtrera efter "Prioritet 1"-taggen, vilket kommer att visa endast poster över alla projekt med den taggen.
