Poster visas i ett bekvämt checklisteformat med kryssrutor för att markera slutförda uppgifter. Du kan scrolla oändligt genom listor som innehåller tusentals poster.

![List](/docs/List.png "List")

Listan tillåter sortering efter postnamn, datum, tilldelad person och mer för att hjälpa till att fokusera på relevanta objekt.

Dessutom kan [filter](/docs/views/filters) tillämpas för att begränsa poster efter tidslinje, tilldelad person, tagg och alla anpassade fält.

## Begränsningar

Även om Listvyn är ett bekvämt verktyg för att hantera poster, har den vissa begränsningar. Poster kan inte skapas direkt i listvyn, och användare kan inte lägga till, ta bort eller ändra ordningen på kolumner. För de som söker en mer kraftfull version av listvyn rekommenderas [Databasvyn](/docs/views/database) eftersom den erbjuder större flexibilitet och funktionalitet.
