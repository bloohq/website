**Vårt uppdrag är att organisera världens arbete.**

Du kan använda Blue för att hantera vad som helst — från ditt försäljnings-CRM till din rekryteringsprocess och allt med repeterbara processer. Blue ger klarhet och skalbarhet till vilken process som helst, så att du håller koll på allt.

Detta är vår dokumentation. Vi har noggrant kartlagt alla funktioner och egenskaper för att möjliggöra din framgång på vår plattform.

Om du har några frågor, tveka inte att kontakta vårt team på <support@blue.cc>

Varje dag tycker vi om att bygga Blue.

Vi hoppas att du kommer att tycka om att använda det.

Fortsätt vara produktiv,

Blue-teamet.
