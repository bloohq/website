Anpassade fältet Enkelvalsval låter dig skapa en rullgardinslista med alternativ som användare kan välja mellan när de matar in data i poster. Du kan definiera upp till 10,000 alternativ med etiketter och färger.

![SingleSelect 1](/docs/SingleSelect_1.png "SingleSelect 1")

Skapa ett anpassat fält för Enkelvalsval

::callout
---

icon: lightbulb
---

Att lägga till alternativ ett efter ett kan bli tråkigt för rullgardinsmenyer. Lyckligtvis kan alternativ läggas till i bulk genom att kopiera och klistra in från en punktlista/radavskild lista eller en kolumn från kalkylblad. Kopiera och klistra in listan med val i dialogrutan för att skapa alternativ i Blue. Detta kommer automatiskt att skapa ett alternativ för varje rad eller punkt, vilket effektiviserar konfigurationen.
::

Ett gränssnitt för sök-medan-du-skriver gör det enkelt att hitta det idealiska valet, även med tusentals alternativ.

![SingleSelect 2](/docs/SingleSelect_2.png "SingleSelect 2")

Sök-medan-du-skriver för att välja ett alternativ

Efter skapandet kan projektadministratörer fortfarande redigera listan med alternativ efter behov, inklusive omordning, tillägg eller borttagningar.

Eftersom bara ett alternativ kan väljas per post fungerar detta fält bra för kategorisk data som produkttyper, statusetiketter, värderingsbetyg, risknivåer och mer.

![SingleSelect 3](/docs/SingleSelect_3.png "SingleSelect 3")

Skapa dina egna anpassade alternativ för att mappa ditt projekt eller process perfekt

Fältet Enkelvalsval levererar en skräddarsydd rullgardinsmeny utformad för snabb val från omfattande fördefinierade alternativ. Att definiera anpassade alternativ och etiketter anpassar fältet nära dina arbetsflöden och terminologi. Dessa rullgardinsmenyer integreras tätt över hela Blue. [Projektadministratörer](/sv/docs/user-management/roles/project-administrator) kan begränsa visnings- och redigeringsåtkomst efter [användarroll](/sv/docs/user-management/roles/custom-user-roles), externa [formulär](/sv/docs/projects/forms) kan använda Enkelvalsval för standardiserad inmatning, och [dashboards](/sv/docs/dashboards/introduction) kan filtrera efter Select-värden. Oavsett om du samlar kategorisk data eller standardiserar terminologi ger fältet Enkelvalsval en optimerad upplevelse för att hantera anpassade rullgardinsmenyer som är anpassade till dina affärsbehov.
