Datum-anpassningsfältet gör det möjligt för dig att associera viktiga datum och tider med poster i Blue. Detta kan hjälpa till att spåra viktiga händelser relaterade till varje post, som startdatum, slutdatum, slutförandetider och mer.

![Date](/docs/Date.png "Date")

Välj exakta datum och tidsintervall

Med den interaktiva datumväljaren kan du enkelt välja exakta datum, tider och intervall — med stöd för tidigare, nuvarande eller framtida datum.

::callout
---

icon: lightbulb
---

Blue stöder för närvarande endast dd/mm/yyyy-format, men vi arbetar på att göra ett användaralternativ för att stödja amerikanska datum (mm/dd/yyyy)
::

Eftersom flera datumfält kan läggas till varje post kan du fånga varje betydelsefullt datum relaterat till en post. Till exempel, i en försäljningskontext kan du använda separata datumfält för att spåra första kontaktdatum, produktdemodatum, förslagssänt datum, kontraktssignerat datum och betalning mottaget datum. Detta ger klarhet i utvecklingen av affärer från första kontakt till avslut.

När de väl har lagts till kan datum filtreras över projekt med Blues avancerade funktioner, vilket möjliggör datumbaserad rapportering. Du kan också valfritt visa anpassade datumfält på projektkalendrar för visuell tidslinjerepresentation [och synkronisera denna data till din personliga eller arbetskalender](/sv/docs/integrations/calendar-sync).
