Blue API tillhandahåller omfattande företagsadministrationsmöjligheter, vilket gör att du kan skapa, fråga och hantera företag inom din Blue-arbetsyta.

## Tillgängliga Operationer

### Skapa Företag
- [Skapa ett Företag](/api/company-management/create-company) - Skapa nya företag med anpassade namn och slugs

### Fråga Företag
- [Fråga Företag](/api/company-management/query-company) - Hämta företagsinformation med hjälp av ID eller slug

## Nyckelkoncept

### Företagsidentifiering
Företag i Blue kan identifieras med hjälp av två metoder:
1. **Företags-ID** - En unik identifierare (CUID) som automatiskt genereras när ett företag skapas
2. **Företags-Slug** - En URL-vänlig identifierare som du anger när du skapar ett företag

### Åtkomstkontroll
- Användare kan endast få åtkomst till företag där de är medlemmar
- Företagsoperationer respekterar användartillstånd och åtkomstnivåer
- Blockerade företag kan inte nås genom API:et

### Faktureringsöverväganden
Varje företag i Blue faktureras separat, så att skapa nya företag kan påverka din kontofakturering.
