API Blue предоставляет обширные возможности управления компаниями, позволяя вам создавать, запрашивать и управлять компаниями в вашем рабочем пространстве Blue.

## Доступные Операции

### Создание Компаний
- [Создать Компанию](/api/company-management/create-company) - Создайте новые компании с пользовательскими именами и слагами

### Запрос Компаний
- [Запросить Компанию](/api/company-management/query-company) - Получите информацию о компании, используя ID или слаг

## Ключевые Концепции

### Идентификация Компании
Компании в Blue могут быть идентифицированы с помощью двух методов:
1. **ID Компании** - Уникальный идентификатор (CUID), автоматически генерируемый при создании компании
2. **Слаг Компании** - Идентификатор, удобный для URL, который вы указываете при создании компании

### Контроль Доступа
- Пользователи могут получать доступ только к компаниям, в которых они являются участниками
- Операции с компаниями учитывают разрешения пользователей и уровни доступа
- Запрещенные компании не могут быть доступны через API

### Учетные Соображения
Каждая компания в Blue выставляется на отдельный счет, поэтому создание новых компаний может повлиять на выставление счета вашему аккаунту.
