Теги — это полезный способ организации вашей работы. Вы можете использовать их для обозначения различных типов задач, срочности или любой категории, которая подходит для вашего проекта. Каждый тег может быть выделен цветом для легкого распознавания и добавления красок в ваше рабочее пространство.

<youtube url="https://www.youtube.com/watch?v=nlyY3cUUpkQ" />


## Использование тегов

- **Добавление тегов**: Чтобы создать тег, подумайте об аспектах ваших задач, которые нуждаются в категоризации — будь то характер задачи, такой как «баг», «back-end» или «UI», или её приоритет. Как только вы определите свои категории, вы можете легко создавать теги и назначать им уникальные цвета для мгновенного распознавания.
- **Фильтрация по тегам**: Возможности фильтрации Blue делают управление вашей рабочей нагрузкой более интуитивным. Вы можете фильтровать теги, используя выпадающий значок фильтра или непосредственно в представлении доски, нажав на нужный тег.

Межпроектные теги <br/>
Для организационных представлений, таких как дашборды и карты, теги становятся особенно важными, поскольку Blue автоматически объединяет теги с одинаковыми названиями. Поэтому вам нужно только отфильтровать по тегу «Приоритет 1», который покажет только записи во всех проектах с этим тегом.
