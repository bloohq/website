Пользовательское поле "Файл" позволяет загружать и прикреплять файлы непосредственно к записям в Blue. Поддерживается практически любой тип файлов, включая документы, таблицы, изображения, PDF и многое другое.

![File 1](/docs/File_1.png "File 1")

Структурированная загрузка файлов обеспечивает видимость связанных файлов для всех

Каждое поле для файлов позволяет загружать несколько файлов размером до 5 ГБ каждый. Загруженные файлы, такие как изображения, PDF и документы Word, могут быть предварительно просмотрены, если тип файла поддерживается. Все пользователи проекта могут скачивать прикрепленные файлы для доступа к соответствующей информации и содержимому записи.

![File 2](/docs/File_2.png "File 2")

Пользователи могут предварительно просматривать и скачивать файлы в пользовательском поле "Файл"

Загруженные файлы доступны на уровне всего проекта во вкладке ["Файлы"](/ru/docs/projects/file-management). Однако пользователи с пользовательскими ролями с ограниченным доступом к записям не увидят эти файлы. Вкладка "Файлы" проекта поддерживает создание папок для организации файлов, но файлы нельзя сортировать в эти папки из самого пользовательского поля.

::callout
---

icon: lightbulb
---

История версий файлов не ведется — новые загрузки не заменяют существующие файлы. Файлы также не включаются в отчеты экспорта CSV.
::

## Интеграция с формами

Пользовательское поле "Файл" [легко интегрируется с формами](/ru/docs/projects/forms), позволяя внешним пользователям загружать файлы при отправке информации. Например, кандидаты могут прикреплять резюме и сопроводительные письма к формам заявок на работу, клиенты могут предоставлять изображения и примеры для запросов творческих проектов, или клиенты могут отправлять документы и документацию через формы приема клиентов. Включая пользовательские поля "Файл" в формы, внешние заинтересованные стороны могут предоставлять важные файлы и активы для обогащения записей, созданных из отправок форм. Это обеспечивает преимущество собранного контента без необходимости создания учетных записей в Blue для внешних пользователей.

## Доступ к файлам через API

Как и любой другой системный файл, файлы, загруженные через пользовательское поле "Файл", могут быть доступны и управляться через [API Blue](/ru/docs/integrations/api). Когда файлы прикрепляются к записям с использованием поля "Файл", эти файлы становятся доступными через стандартные вызовы API, связанные с файлами. Разработчики могут использовать полный набор возможностей API для программного скачивания, загрузки, организации и интеграции файлов из пользовательского поля "Файл". Будь то автоматическая обработка загруженных активов или добавление метаданных, API обеспечивает бесшовную интеграцию, предоставляя единый доступ ко всем добавленным пользователями файлам в Blue.

Прикрепляя часто используемые документы и активы, поля "Файл" исключают поиск по нескольким местам для нахождения контента, связанного с записью. Это обеспечивает быстрый, централизованный доступ к нужным файлам.
