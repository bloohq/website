Настраиваемое поле "Одиночный выбор" позволяет создать раскрывающийся список опций, из которых пользователи могут выбирать при вводе данных в записи. Вы можете определить до 10 000 опций с метками и цветами.

![SingleSelect 1](/docs/SingleSelect_1.png "SingleSelect 1")

Создание настраиваемого поля "Одиночный выбор"

::callout
---

icon: lightbulb
---

Добавление опций по одной может стать утомительным для раскрывающихся списков. К счастью, опции можно добавлять массово, копируя и вставляя из маркированного/разделенного строками списка или столбца электронной таблицы. Скопируйте и вставьте список вариантов в диалог создания опций в Blue. Это автоматически создаст опцию для каждой строки или пункта списка, упростив настройку.
::

Интерфейс поиска при вводе делает поиск идеального выбора легким, даже среди тысяч опций.

![SingleSelect 2](/docs/SingleSelect_2.png "SingleSelect 2")

Поиск при вводе для выбора опции

После создания администраторы проекта могут по-прежнему редактировать список опций по мере необходимости, включая переупорядочивание, добавления или удаления.

Поскольку на запись можно выбрать только одну опцию, это поле хорошо подходит для категориальных данных, таких как типы продуктов, метки статуса, оценочные классы, уровни риска и многое другое.

![SingleSelect 3](/docs/SingleSelect_3.png "SingleSelect 3")

Создайте свои собственные настраиваемые опции для идеального соответствия вашему проекту или процессу

Поле "Одиночный выбор" предоставляет индивидуально настроенный раскрывающийся список, предназначенный для быстрого выбора из обширных предопределенных опций. Определение настроенных опций и меток тесно увязывает поле с вашими рабочими процессами и терминологией. Эти раскрывающиеся списки тесно интегрированы в Blue. [Администраторы проекта](/ru/docs/user-management/roles/project-administrator) могут ограничить доступ к просмотру и редактированию по [роли пользователя](/ru/docs/user-management/roles/custom-user-roles), внешние [формы](/ru/docs/projects/forms) могут использовать одиночный выбор для стандартизированного ввода, а [панели управления](/ru/docs/dashboards/introduction) могут фильтровать по значениям выбора. Будь то сбор категориальных данных или стандартизация терминологии, поле "Одиночный выбор" обеспечивает оптимизированный опыт управления настроенными раскрывающимися списками, согласованными с потребностями вашего бизнеса.
