Пользовательское поле Номер Телефона позволяет хранить числовые номера телефонов для записей в ваших проектах Blue. Это может помочь централизовать важную контактную информацию для клиентов, поставщиков или других объектов в ваших записях.

![PhoneNumber 1](/docs/PhoneNumber_1.png "PhoneNumber 1")

При добавлении номера телефона поле автоматически проверит длину на основе ожидаемого формата для страны. Например, американский номер будет проверен на 10 цифр. Номера сохраняются без какого-либо форматирования, такого как тире или скобки.

Поле номера телефона также поддерживает расширения. При добавлении расширения есть выпадающее меню со всеми странами и их ожидаемыми форматами расширений. Вы можете искать в этом выпадающем меню по номеру расширения или названию страны, чтобы быстро найти и выбрать подходящий формат расширения. Это позволяет установить формат расширения вручную на основе конкретной страны.

![PhoneNumber 2](/docs/PhoneNumber_2.png "PhoneNumber 2")

Поиск кодов стран прямо в Blue

Ключевой особенностью поля Номер Телефона является встроенная функциональность щелчка для звонка. Наведение курсора на сохраненный номер телефона покажет кликабельную иконку телефона. Щелчок по этой иконке автоматически наберет номер, используя ваше приложение или сервис звонков по умолчанию на вашем устройстве. Этот бесшовный опыт позволяет инициировать звонки вашим контактам прямо из Blue.

![PhoneNumber 3](/docs/PhoneNumber_3.png "PhoneNumber 3")

Каждая запись может хранить один номер телефона. Поле не различает между мобильными и стационарными номерами. Фокус направлен на захват основного номера для этого конкретного объекта или контакта.
::

Поле Номер Телефона обеспечивает оптимизированный опыт для легкого хранения и доступа к важной контактной информации из ваших записей. Автоматизированная проверка, щелчок для звонка и упрощенный дизайн одного номера позволяют подключаться к вашим контактам всего одним щелчком.
