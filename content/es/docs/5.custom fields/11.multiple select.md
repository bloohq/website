---

El campo personalizado de Selección Múltiple funciona de manera idéntica al [campo personalizado de Selección Simple](/es/docs/custom-fields/single-select), con una distinción clave: la capacidad de realizar múltiples selecciones por registro.

El campo personalizado de Selección Múltiple le permite crear una lista desplegable de opciones que los usuarios pueden elegir cuando ingresan datos en los registros. Puede definir hasta 10,000 opciones con etiquetas y colores.

![MultipleSelect 1](/docs/MultipleSelect_1.png "MultipleSelect 1")

Elección de colores para opciones dentro de un campo personalizado de Selección Múltiple

Permitir múltiples selecciones es ideal para capturar:

- Taxonomías - Clasificar registros en múltiples categorías (es decir, etiquetar registros con todas las etiquetas relevantes)
- Datos Compuestos - Combinar opciones para describir registros complejos (es decir, seleccionar todas las propiedades de producto aplicables)
- Pistas de Auditoría - Rastrear cambios a lo largo del tiempo (es decir, registrar actualizaciones de estado en un registro)

::callout
---

icon: lightbulb
---

Agregar opciones una por una puede volverse tedioso para menús desplegables de selección. Afortunadamente, las opciones se pueden agregar en lote copiando y pegando desde una lista con viñetas/separada por líneas o una columna de hoja de cálculo. Simplemente copie la lista de opciones y péguela en el diálogo de creación de opciones en Blue. Esto creará automáticamente una opción para cada línea o viñeta, agilizando la configuración.
::

La flexibilidad de las selecciones múltiples amplía cómo los menús desplegables personalizables pueden estructurar los datos del proyecto.

Una interfaz de búsqueda mientras escribe hace que encontrar la selección ideal sea fácil, incluso con miles de opciones.

![MultipleSelect 2](/docs/MultipleSelect_2.png "MultipleSelect 2")

Después de la creación, los administradores del proyecto aún pueden editar la lista de opciones según sea necesario, incluyendo reordenamiento, adiciones o eliminaciones.

El campo de Selección Múltiple ofrece un menú desplegable personalizado diseñado para la selección rápida de opciones predefinidas extensas. Definir opciones y etiquetas personalizadas alinea el campo estrechamente con sus flujos de trabajo y terminología. Estos menús desplegables se integran estrechamente en toda Blue. Los [Administradores de Proyecto](/es/docs/user-management/roles/project-administrator) pueden restringir el acceso de visualización y edición por [rol de usuario](/es/docs/user-management/roles/custom-user-roles), los [formularios](/es/docs/projects/forms) externos pueden aprovechar las Selecciones Múltiples para entrada estandarizada, y los [tableros](/es/docs/dashboards) pueden filtrar por valores de Selección. Ya sea recopilando datos categóricos o estandarizando terminología, el campo de Selección Múltiple proporciona una experiencia optimizada para gestionar menús desplegables personalizados alineados con sus necesidades comerciales.
