El campo personalizado de Selección Individual le permite crear una lista desplegable de opciones que los usuarios pueden elegir al ingresar datos en registros. Puede definir hasta 10,000 opciones con etiquetas y colores.

![SingleSelect 1](/docs/SingleSelect_1.png "SingleSelect 1")

Creando un campo personalizado de Selección Individual

::callout
---

icon: lightbulb
---

Agregar opciones una por una puede volverse tedioso para listas desplegables de selección. Afortunadamente, las opciones se pueden agregar en lotes copiando y pegando desde una lista con viñetas/separada por líneas o una columna de hoja de cálculo. Copie y pegue la lista de opciones en el diálogo de creación de opciones en Blue. Esto creará automáticamente una opción para cada línea o viñeta, agilizando la configuración.
::

Una interfaz de búsqueda mientras escribe hace que encontrar la selección ideal sea fácil, incluso con miles de opciones.

![SingleSelect 2](/docs/SingleSelect_2.png "SingleSelect 2")

Búsqueda mientras escribe para seleccionar una opción

Después de la creación, los administradores del proyecto aún pueden editar la lista de opciones según sea necesario, incluyendo reordenamiento, adiciones o eliminaciones.

Dado que solo se puede seleccionar una opción por registro, este campo funciona bien para datos categóricos como tipos de productos, etiquetas de estado, calificaciones de evaluación, niveles de riesgo y más.

![SingleSelect 3](/docs/SingleSelect_3.png "SingleSelect 3")

Cree sus propias opciones personalizadas para mapear su proyecto o proceso perfectamente

El campo de Selección Individual ofrece una lista desplegable personalizada diseñada para la selección rápida de opciones predefinidas extensas. Definir opciones y etiquetas personalizadas alinea el campo estrechamente con sus flujos de trabajo y terminología. Estas listas desplegables se integran estrechamente en toda la plataforma Blue. Los [Administradores de Proyecto](/es/docs/user-management/roles/project-administrator) pueden restringir el acceso de visualización y edición por [rol de usuario](/es/docs/user-management/roles/custom-user-roles), los [formularios](/es/docs/projects/forms) externos pueden aprovechar las Selecciones Individuales para entrada estandarizada, y los [tableros](/es/docs/dashboards/introduction) pueden filtrar por valores de Selección. Ya sea recopilando datos categóricos o estandarizando terminología, el campo de Selección Individual proporciona una experiencia optimizada para gestionar listas desplegables personalizadas alineadas con sus necesidades comerciales.
