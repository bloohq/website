O campo personalizado de Seleção Única permite criar uma lista suspensa de opções que os usuários podem escolher ao inserir dados em registros. Você pode definir até 10.000 opções com rótulos e cores.

![SingleSelect 1](/docs/SingleSelect_1.png "SingleSelect 1")

Criando um campo personalizado de Seleção Única

::callout
---

icon: lightbulb
---

Adicionar opções uma por uma pode se tornar tedioso para listas suspensas de seleção. Felizmente, as opções podem ser adicionadas em massa copiando e colando de uma lista com marcadores/separada por linhas ou coluna de planilha. Copie e cole a lista de opções no diálogo de criação de opções no Blue. Isso criará automaticamente uma opção para cada linha ou marcador, simplificando a configuração.
::

Uma interface de pesquisa conforme você digita torna fácil encontrar a seleção ideal, mesmo com milhares de opções.

![SingleSelect 2](/docs/SingleSelect_2.png "SingleSelect 2")

Pesquise conforme você digita para selecionar uma opção

Após a criação, os administradores do projeto ainda podem editar a lista de opções conforme necessário, incluindo reordenação, adições ou remoções.

Como apenas uma opção pode ser selecionada por registro, este campo funciona bem para dados categóricos como tipos de produto, rótulos de status, notas de avaliação, níveis de risco e mais.

![SingleSelect 3](/docs/SingleSelect_3.png "SingleSelect 3")

Crie suas próprias opções personalizadas para mapear seu projeto ou processo perfeitamente

O campo de Seleção Única oferece uma lista suspensa personalizada projetada para seleção rápida de extensas opções predefinidas. Definir opções e rótulos personalizados alinha o campo intimamente com seus fluxos de trabalho e terminologia. Essas listas suspensas se integram perfeitamente em todo o Blue. [Administradores de Projeto](/pt/docs/user-management/roles/project-administrator) podem restringir o acesso de visualização e edição por [função do usuário](/pt/docs/user-management/roles/custom-user-roles), [formulários](/pt/docs/projects/forms) externos podem aproveitar Seleções Únicas para entrada padronizada, e [painéis](/pt/docs/dashboards/introduction) podem filtrar por valores de Seleção. Seja coletando dados categóricos ou padronizando terminologia, o campo de Seleção Única fornece uma experiência otimizada para gerenciar listas suspensas personalizadas alinhadas com suas necessidades de negócio.
