Tags são uma forma útil de organizar seu trabalho. Você pode usá-las para mostrar diferentes tipos de tarefas, urgência, ou qualquer categoria que se adapte ao seu projeto. Cada tag pode ser codificada por cor para fácil identificação e para adicionar cor ao seu workspace.

<youtube url="https://www.youtube.com/watch?v=nlyY3cUUpkQ" />


## Usando Tags

- **Adicionando Tags**: Para criar uma tag, considere os aspectos das suas tarefas que precisam de categorização—seja a natureza da tarefa, como 'bug', 'back-end', ou 'UI', ou sua prioridade. Uma vez que você tenha identificado suas categorias, pode facilmente criar tags e atribuir cores únicas para reconhecimento imediato.
- **Filtrando com Tags**: As capacidades de filtragem do Blue tornam o gerenciamento da sua carga de trabalho mais intuitivo. Você pode filtrar tags usando o ícone de filtro dropdown ou diretamente na visualização do board clicando na tag desejada.

Tagging entre Projetos <br/>
Para visualizações de toda a organização, como Dashboards e mapas, a marcação com tags torna-se especialmente importante porque o Blue automaticamente mescla tags com o mesmo nome. Então você só precisa filtrar pela tag "Prioridade 1", que mostrará apenas registros de todos os projetos com essa tag.
