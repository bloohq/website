---
title: Copiando Registros
description: Você pode copiar registros dentro do mesmo projeto ou entre projetos.
---

Blue permite que você duplique registros dentro de um projeto ou entre projetos facilmente. Isso pode ajudar a acelerar seu fluxo de trabalho replicando informações sem ter que reescrevê-las.

Para copiar um registro:

1. Navegue até o registro que deseja copiar
2. Clique no menu "..."
3. Selecione "Copiar"
4. Escolha se deseja copiar para o projeto atual ou outro projeto
5. A cópia será criada com os mesmos detalhes.

![Copying Record 1](/docs/CopyingRecord.png "Copying Record 1")

Pontos importantes:

- Ao copiar entre projetos, quaisquer anexos e comentários serão copiados
- Campos personalizados serão copiados para o novo projeto, e o novo projeto também terá os campos personalizados adicionais
- Se o projeto de destino tiver o mesmo tipo de campo personalizado com o mesmo nome, os dados serão automaticamente correspondidos
- Apenas administradores e membros da equipe podem mover registros
