Field kustom Single Select memungkinkan Anda membuat daftar dropdown berisi pilihan yang dapat dipilih pengguna saat memasukkan data ke dalam record. Anda dapat menentukan hingga 10.000 pilihan dengan label dan warna.

![SingleSelect 1](/docs/SingleSelect_1.png "SingleSelect 1")

Membuat field kustom Single Select

::callout
---

icon: lightbulb
---

Menambahkan pilihan satu per satu bisa menjadi membosankan untuk dropdown select. Untungnya, pilihan dapat ditambahkan secara massal dengan menyalin dan menempel dari daftar berpoin/terpisah baris atau kolom spreadsheet. Salin dan tempel daftar pilihan ke dalam dialog pembuatan pilihan di Blue. Ini akan secara otomatis membuat pilihan untuk setiap baris atau poin, menyederhanakan konfigurasi.
::

Antarmuka pencarian saat mengetik memudahkan menemukan pilihan yang ideal, bahkan dengan ribuan pilihan.

![SingleSelect 2](/docs/SingleSelect_2.png "SingleSelect 2")

Pencarian saat mengetik untuk memilih pilihan

Setelah dibuat, administrator proyek masih dapat mengedit daftar pilihan sesuai kebutuhan, termasuk pengurutan ulang, penambahan, atau penghapusan.

Karena hanya satu pilihan yang dapat dipilih per record, field ini bekerja dengan baik untuk data kategorikal seperti jenis produk, label status, tingkat penilaian, tingkat risiko, dan lainnya.

![SingleSelect 3](/docs/SingleSelect_3.png "SingleSelect 3")

Buat pilihan kustom Anda sendiri untuk memetakan proyek atau proses Anda dengan sempurna

Field Single Select memberikan dropdown yang disesuaikan dan dirancang untuk pemilihan cepat dari pilihan predefined yang ekstensif. Menentukan pilihan dan label yang disesuaikan menyelaraskan field dengan alur kerja dan terminologi Anda. Dropdown ini terintegrasi erat di seluruh Blue. [Administrator Proyek](/id/docs/user-management/roles/project-administrator) dapat membatasi akses tampilan dan edit berdasarkan [peran pengguna](/id/docs/user-management/roles/custom-user-roles), [formulir](/id/docs/projects/forms) eksternal dapat memanfaatkan Single Select untuk input yang distandarisasi, dan [dashboard](/id/docs/dashboards/introduction) dapat memfilter berdasarkan nilai Select. Baik untuk mengumpulkan data kategorikal atau menstandarisasi terminologi, field Single Select menyediakan pengalaman yang dioptimalkan untuk mengelola dropdown yang disesuaikan selaras dengan kebutuhan bisnis Anda.
