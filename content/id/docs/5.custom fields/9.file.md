Bidang kustom File memungkinkan pengunggahan dan melampirkan berkas secara langsung ke catatan dalam Blue. Hampir semua jenis berkas didukung, termasuk dokumen, spreadsheet, gambar, PDF, dan lainnya.

![File 1](/docs/File_1.png "File 1")

Pengunggahan berkas terstruktur memastikan bahwa semua orang memiliki visibilitas pada berkas terkait

Setiap bidang file memungkinkan pengunggahan beberapa berkas, dengan setiap berkas berukuran hingga 5GB. Berkas yang diunggah, seperti gambar, PDF, dan dokumen Word, dapat dipratinjau jika jenis berkas tersebut didukung. Semua pengguna proyek dapat mengunduh berkas yang dilampirkan untuk mengakses informasi dan konten catatan yang relevan.

![File 2](/docs/File_2.png "File 2")

Pengguna dapat mempratinjau dan mengunduh berkas dalam bidang kustom File

Berkas yang diunggah dibagikan ke seluruh proyek di bawah ['tab File'](/id/docs/projects/file-management). Namun, peran pengguna kustom dengan akses catatan terbatas tidak akan melihat berkas tersebut. Tab File proyek mendukung pembuatan folder untuk mengorganisir berkas, tetapi berkas tidak dapat disortir ke dalam folder ini dari dalam bidang kustom itu sendiri.

## Integrasi Formulir

Bidang kustom File [terintegrasi secara seamless dengan Formulir](/id/docs/projects/forms), memungkinkan pengguna eksternal untuk mengunggah berkas saat mengirimkan informasi. Misalnya, kandidat dapat melampirkan CV dan surat lamaran ke formulir aplikasi pekerjaan, pelanggan dapat menyediakan gambar dan contoh untuk permintaan proyek kreatif, atau klien dapat mengirimkan dokumen dan dokumentasi melalui formulir intake klien. Dengan menyertakan bidang kustom File pada formulir, pemangku kepentingan eksternal dapat menyediakan berkas dan aset penting untuk memperkaya catatan yang dihasilkan dari pengiriman formulir. Ini memberikan manfaat konten yang dikumpulkan tanpa mengharuskan pengguna eksternal memiliki akun dalam Blue.

## Akses Berkas API

Seperti berkas sistem lainnya, berkas yang diunggah melalui bidang kustom File dapat diakses dan dikelola melalui [Blue API](/id/docs/integrations/api). Ketika berkas dilampirkan ke catatan menggunakan bidang File, berkas tersebut menjadi tersedia melalui panggilan API terkait berkas standar. Pengembang dapat memanfaatkan rangkaian lengkap kemampuan API untuk secara programatis mengunduh, mengunggah, mengorganisir, dan mengintegrasikan berkas dari bidang kustom File. Baik secara otomatis memproses aset yang diunggah atau menambahkan metadata, API memungkinkan integrasi yang seamless dengan menyediakan akses terpadu ke semua berkas yang ditambahkan pengguna dalam Blue.

Dengan melampirkan dokumen dan aset yang sering digunakan, bidang File menghilangkan pencarian melalui beberapa lokasi untuk menemukan konten yang terkait dengan catatan. Ini memberikan akses cepat dan terpusat ke berkas yang Anda butuhkan.
