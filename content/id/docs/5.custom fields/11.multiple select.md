Field kustom Multiple Select berfungsi identik dengan [field kustom Single Select](/id/docs/custom-fields/single-select), dengan satu perbedaan utama â€” kemampuan untuk melakukan beberapa pilihan per record.

Field kustom Multiple Select memungkinkan Anda membuat daftar dropdown dengan opsi yang dapat dipilih pengguna saat memasukkan data ke dalam record. Anda dapat menentukan hingga 10.000 opsi dengan label dan warna.

![MultipleSelect 1](/docs/MultipleSelect_1.png "MultipleSelect 1")

Memilih warna untuk opsi dalam field kustom Multiple Select

Memungkinkan beberapa pilihan ideal untuk menangkap:

- Taksonomi - Mengklasifikasikan record ke dalam beberapa kategori (misalnya menandai record dengan semua label yang relevan)
- Data Komposit - Menggabungkan opsi untuk mendeskripsikan record kompleks (misalnya memilih semua properti produk yang berlaku)
- Audit Trail - Melacak perubahan dari waktu ke waktu (misalnya mencatat pembaruan status ke record)

::callout
---

icon: lightbulb
---

Menambahkan opsi satu per satu dapat menjadi membosankan untuk dropdown select. Untungnya, opsi dapat ditambahkan secara massal dengan menyalin dan menempel dari daftar berbutir/terpisah baris atau kolom spreadsheet. Cukup salin daftar pilihan dan tempel ke dalam dialog pembuatan opsi di Blue. Ini akan secara otomatis membuat opsi untuk setiap baris atau poin bullet, menyederhanakan konfigurasi.
::

Fleksibilitas beberapa pilihan memperluas bagaimana dropdown yang dapat disesuaikan dapat menyusun data proyek.

Antarmuka pencarian-sambil-mengetik membuat menemukan pilihan ideal menjadi mudah, bahkan dengan ribuan opsi.

![MultipleSelect 2](/docs/MultipleSelect_2.png "MultipleSelect 2")

Setelah dibuat, administrator proyek masih dapat mengedit daftar opsi sesuai kebutuhan, termasuk pengurutan ulang, penambahan, atau penghapusan.

Field Multiple Select memberikan dropdown yang disesuaikan yang dirancang untuk pemilihan cepat dari opsi yang telah ditentukan secara ekstensif. Menentukan opsi dan label yang disesuaikan menyelaraskan field dengan alur kerja dan terminologi Anda. Dropdown ini terintegrasi erat di seluruh Blue. [Administrator Proyek](/id/docs/user-management/roles/project-administrator) dapat membatasi akses tampilan dan edit berdasarkan [peran pengguna](/id/docs/user-management/roles/custom-user-roles), [formulir](/id/docs/projects/forms) eksternal dapat memanfaatkan Multiple Select untuk input standar, dan [dashboard](/id/docs/dashboards) dapat memfilter berdasarkan nilai Select. Baik mengumpulkan data kategorikal atau menstandardisasi terminologi, field Multiple Select menyediakan pengalaman yang dioptimalkan untuk mengelola dropdown yang disesuaikan selaras dengan kebutuhan bisnis Anda.
