---
title: 公司管理
description: Blue API 中公司管理操作的概述。
---

Blue API 提供全面的公司管理功能，允許您在您的 Blue 工作區內創建、查詢和管理公司。

## 可用操作

### 創建公司
- [創建公司](/api/company-management/create-company) - 使用自定義名稱和標識符創建新公司

### 查詢公司
- [查詢公司](/api/company-management/query-company) - 使用 ID 或標識符檢索公司信息

## 關鍵概念

### 公司識別
Blue 中的公司可以使用兩種方法進行識別：
1. **公司 ID** - 在創建公司時自動生成的唯一標識符 (CUID)
2. **公司標識符** - 在創建公司時指定的 URL 友好標識符

### 訪問控制
- 用戶只能訪問他們是成員的公司
- 公司操作遵循用戶權限和訪問級別
- 被禁止的公司無法通過 API 訪問

### 計費考量
Blue 中的每家公司都是單獨計費的，因此創建新公司可能會影響您的帳戶計費。
