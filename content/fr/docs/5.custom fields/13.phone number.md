Le champ personnalisé Numéro de Téléphone vous permet de stocker des numéros de téléphone numériques pour les enregistrements dans vos projets Blue. Cela peut aider à centraliser les informations de contact importantes pour les clients, fournisseurs ou autres entités dans vos enregistrements.

![PhoneNumber 1](/docs/PhoneNumber_1.png "PhoneNumber 1")

Lors de l'ajout d'un numéro de téléphone, le champ validera automatiquement la longueur basée sur le format attendu pour le pays. Par exemple, un numéro américain sera validé à 10 chiffres. Les numéros sont stockés sans aucun formatage, comme les tirets ou les parenthèses.

Le champ numéro de téléphone prend également en charge les extensions. Lors de l'ajout d'une extension, il y a un menu déroulant avec tous les pays et leurs formats d'extension attendus. Vous pouvez rechercher dans ce menu déroulant par numéro d'extension ou nom de pays pour trouver rapidement et sélectionner le format d'extension approprié. Cela vous permet de définir le format d'extension manuellement en fonction du pays spécifique.

![PhoneNumber 2](/docs/PhoneNumber_2.png "PhoneNumber 2")

Recherchez les codes de pays directement dans Blue

Une fonctionnalité clé du champ Numéro de Téléphone est la fonctionnalité intégrée de clic pour appeler. Survoler un numéro de téléphone stocké révélera une icône de téléphone cliquable. Cliquer sur cette icône composera automatiquement le numéro en utilisant votre application ou service d'appel par défaut sur votre appareil. Cette expérience transparente vous permet d'initier des appels vers vos contacts directement depuis Blue.

![PhoneNumber 3](/docs/PhoneNumber_3.png "PhoneNumber 3")

Chaque enregistrement peut stocker un numéro de téléphone. Le champ ne différencie pas entre les numéros mobiles et fixes. L'accent est mis sur la capture du numéro principal pour cette entité ou contact particulier.
::

Le champ Numéro de Téléphone offre une expérience optimisée pour stocker facilement et accéder aux informations de contact importantes de vos enregistrements. La validation automatisée, le clic pour appeler et une conception rationalisée à numéro unique vous permettent de vous connecter avec vos contacts en un seul clic.
