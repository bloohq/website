Le champ personnalisé Sélection Multiple fonctionne de manière identique au [champ personnalisé Sélection Simple](/fr/docs/custom-fields/single-select), avec une distinction clé — la possibilité de faire plusieurs sélections par enregistrement.

Le champ personnalisé Sélection Multiple vous permet de créer une liste déroulante d'options que les utilisateurs peuvent choisir lors de la saisie de données dans les enregistrements. Vous pouvez définir jusqu'à 10 000 options avec des libellés et des couleurs.

![MultipleSelect 1](/docs/MultipleSelect_1.png "MultipleSelect 1")

Choix des couleurs pour les options dans un champ personnalisé Sélection Multiple

Permettre plusieurs sélections est idéal pour capturer :

- Taxonomies - Classer les enregistrements dans plusieurs catégories (c'est-à-dire étiqueter les enregistrements avec tous les libellés pertinents)
- Données Composites - Combiner des options pour décrire des enregistrements complexes (c'est-à-dire sélectionner toutes les propriétés de produit applicables)
- Pistes d'Audit - Suivre les changements au fil du temps (c'est-à-dire enregistrer les mises à jour de statut d'un enregistrement)

::callout
---

icon: lightbulb
---

Ajouter des options une par une peut devenir fastidieux pour les listes déroulantes de sélection. Heureusement, les options peuvent être ajoutées en masse en copiant et collant depuis une liste à puces/séparée par des lignes ou une colonne de feuille de calcul. Il suffit de copier la liste de choix et de les coller dans la boîte de dialogue de création d'options dans Blue. Cela créera automatiquement une option pour chaque ligne ou puce, rationalisant la configuration.
::

La flexibilité des sélections multiples étend la façon dont les listes déroulantes personnalisables peuvent structurer les données de projet.

Une interface de recherche au fur et à mesure de la frappe rend facile la recherche de la sélection idéale, même avec des milliers d'options.

![MultipleSelect 2](/docs/MultipleSelect_2.png "MultipleSelect 2")

Après création, les administrateurs de projet peuvent encore modifier la liste d'options selon les besoins, y compris la réorganisation, les ajouts ou les suppressions.

Le champ Sélection Multiple offre une liste déroulante personnalisée conçue pour une sélection rapide parmi de nombreuses options prédéfinies. Définir des options et des libellés personnalisés aligne étroitement le champ avec vos flux de travail et votre terminologie. Ces listes déroulantes s'intègrent étroitement dans Blue. Les [Administrateurs de Projet](/fr/docs/user-management/roles/project-administrator) peuvent restreindre l'accès en visualisation et modification par [rôle utilisateur](/fr/docs/user-management/roles/custom-user-roles), les [formulaires](/fr/docs/projects/forms) externes peuvent exploiter les Sélections Multiples pour une saisie standardisée, et les [tableaux de bord](/fr/docs/dashboards) peuvent filtrer par valeurs de Sélection. Que vous collectiez des données catégorielles ou standardisiez la terminologie, le champ Sélection Multiple fournit une expérience optimisée pour gérer les listes déroulantes personnalisées alignées avec vos besoins métier.
