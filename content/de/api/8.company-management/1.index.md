Die Blue API bietet umfassende Funktionen zur Unternehmensverwaltung, mit denen Sie Unternehmen in Ihrem Blue-Arbeitsbereich erstellen, abfragen und verwalten können.

## Verfügbare Operationen

### Unternehmen erstellen
- [Ein Unternehmen erstellen](/api/company-management/create-company) - Erstellen Sie neue Unternehmen mit benutzerdefinierten Namen und Slugs

### Unternehmen abfragen
- [Unternehmen abfragen](/api/company-management/query-company) - Abrufen von Unternehmensinformationen mit ID oder Slug

## Schlüsselkonzepte

### Unternehmensidentifikation
Unternehmen in Blue können auf zwei Arten identifiziert werden:
1. **Unternehmens-ID** - Ein eindeutiger Identifikator (CUID), der automatisch beim Erstellen eines Unternehmens generiert wird
2. **Unternehmens-Slug** - Ein URL-freundlicher Identifikator, den Sie beim Erstellen eines Unternehmens angeben

### Zugriffskontrolle
- Benutzer können nur auf Unternehmen zugreifen, in denen sie Mitglieder sind
- Unternehmensoperationen respektieren die Benutzerberechtigungen und Zugriffslevel
- Gesperrte Unternehmen können nicht über die API zugegriffen werden

### Abrechnungsüberlegungen
Jedes Unternehmen in Blue wird separat abgerechnet, sodass die Erstellung neuer Unternehmen Ihre Kontenabrechnung beeinflussen kann.
