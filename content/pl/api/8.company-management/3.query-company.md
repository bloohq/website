---
title: Zapytanie o Firmę
description: Jak uzyskać informacje o firmie za pomocą ID lub slug z API Blue.
---

Aby uzyskać informacje o firmie, możesz użyć zapytania `company` z identyfikatorem firmy lub slug:

## Zapytanie według ID Firmy

```graphql
query GetCompanyById {
  company(id: "company-id-here") {
    id
    name
    slug
    createdAt
    updatedAt
  }
}
```

## Zapytanie według Sluga Firmy

To samo zapytanie akceptuje również slug firmy, co ułatwia uzyskanie informacji o firmie za pomocą identyfikatora przyjaznego URL:

```graphql
query GetCompanyBySlug {
  company(id: "company-slug-here") {
    id
    name
    slug
    createdAt
    updatedAt
  }
}
```

## Przykład Odpowiedzi

Oba zapytania zwrócą ten sam obiekt firmy:

```json
{
  "data": {
    "company": {
      "id": "cuid123456789",
      "name": "Acme Corporation",
      "slug": "acme-corp",
      "createdAt": "2024-01-15T10:30:00.000Z",
      "updatedAt": "2024-03-20T14:45:00.000Z"
    }
  }
}
```

## Ważne Uwagi

- Parametr `id` akceptuje zarówno ID firm, jak i slug
- Zwracane będą tylko firmy, w których obecny użytkownik jest członkiem
- Jeśli firma nie zostanie znaleziona lub użytkownik nie ma dostępu, zostanie zgłoszony błąd `CompanyNotFoundError`
- Zbanowane firmy zgłoszą błąd `CompanyBannedError`

## Dostępne Pola

Możesz zapytać o dodatkowe pola firmy w razie potrzeby:

```graphql
query GetCompanyDetails {
  company(id: "company-slug") {
    id
    name
    slug
    # Add more fields as needed based on your requirements
    # Check the GraphQL schema for all available fields
  }
}
```
