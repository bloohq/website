description: Przegląd operacji zarządzania firmą w API Blue.
---

API Blue oferuje kompleksowe możliwości zarządzania firmami, umożliwiając tworzenie, zapytania i zarządzanie firmami w Twoim środowisku Blue.

## Dostępne Operacje

### Tworzenie Firm
- [Utwórz Firmę](/api/company-management/create-company) - Twórz nowe firmy z niestandardowymi nazwami i slugami

### Zapytania o Firmy
- [Zapytaj o Firmę](/api/company-management/query-company) - Pobierz informacje o firmie za pomocą ID lub sluga

## Kluczowe Pojęcia

### Identyfikacja Firmy
Firmy w Blue można identyfikować na dwa sposoby:
1. **ID Firmy** - Unikalny identyfikator (CUID) automatycznie generowany podczas tworzenia firmy
2. **Slug Firmy** - Przyjazny dla URL identyfikator, który określasz podczas tworzenia firmy

### Kontrola Dostępu
- Użytkownicy mogą uzyskiwać dostęp tylko do firm, w których są członkami
- Operacje na firmach respektują uprawnienia użytkowników i poziomy dostępu
- Zbanowane firmy nie mogą być dostępne przez API

### Rozważania dotyczące Rozliczeń
Każda firma w Blue jest rozliczana oddzielnie, więc tworzenie nowych firm może wpłynąć na rozliczenia Twojego konta.
